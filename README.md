# NHANES Smoking and Hypertension Study

## Objective
This project investigates the relationship between smoking and hypertension using real-world observational data from NHANES.

## Methods
- Logistic regression
- Propensity score weighting to adjust for confounding
- Exploratory data analysis

## Structure
- `data/` : raw data files
- `notebooks/` : analysis notebook
- `results/` : outputs and figures

## Status
Project setup completed. Analysis in progress.
